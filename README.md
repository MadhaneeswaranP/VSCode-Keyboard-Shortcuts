# VSCode-Keyboard-Shortcuts

> VSCode keyboard shortcuts for improving productivity

## Official List of all commands

* [Windows](https://code.visualstudio.com/shortcuts/keyboard-shortcuts-windows.pdf)
* [Mac](https://code.visualstudio.com/shortcuts/keyboard-shortcuts-macos.pdf)

## Open/View

Open Command Pallete

```bash
Shift+Cmd+P
Shift+Ctrl+P
```

Access Settings

```bash
Cmd+,
Ctrl+,
```

Toggle Terminal

```bash
Ctrl+`
Ctrl+`
```

Create New Terminal
```
Shift+Ctrl+`
Shift+Ctrl+`
```

Toggle Sidebar

```bash
Cmd+B 
Ctrl+B
```

Open New Window/Instance

```bash
Shift+Cmd+N 
Shift+Ctrl+N
```

Close Window

```bash
Cmd+W 
Ctrl+W
```

Change File Tabs

```bash
Cmd+Tab 
Ctrl+Tab
```


Zoom

```bash
Cmd++ # Zoom in
Cmd+- # Zoom out

Ctrl++ # Zoom in
Ctrl+- # Zoom out
```

Spilt Editor

```bash
Cmd+\
Ctrl+\
```

Add Multiple Cursor

```bash
Option+Click
Alt+Click
```
